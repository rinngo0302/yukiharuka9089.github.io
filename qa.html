Q&A

Q.参加できません。
A.チャンネルコードが間違えている可能性があります。
もしくは、ブラウザが対応していない可能性もあります。その場合は、こちらへどうぞ



Q.チャンネルコードとは何ですか？
A.ゲームをするために必要なコードです。
ホストとなる人がチャンネルコードを設定をするので、それと同じチャンネルコードを入力してください。



Q.違うところに参加をしてしまいます。
A.チャンネルコードを間違えてしまった可能性が高いです。
または、同じチャンネルコードが二つ以上あるのかもしれません。出来るだけ、他と被らないようにホストの方はチャンネルコードを設定してください。



Q.micro:bitに接続ができません。
A.恐らく、使用しているブラウザが対応していない可能性があります。
Google Chrome や Microsoft Edge でアクセスすると接続することができます。

また、Google Chrome や Microsoft Edge を使っているのに接続できないという人は、バージョンが低い可能性があります。
最新バージョンをDLしてください。
